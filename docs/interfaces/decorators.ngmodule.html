Optional providers

providers: any[]

Inherited from IModuleSettings.providers

  • Defined in interfaces/interfaces.ts:44

Defines the set of injectable objects that are available in the injector of this module.

Simple Example

Here is an example of a class that can be injected:

class Greeter {
   greet(name:string) {
     return 'Hello ' + name + '!';
   }
}

 @NgModule({
  providers: [
    Greeter
  ]
})
class HelloWorld {
  greeter:Greeter;

  constructor(greeter:Greeter) {
    this.greeter = greeter;
  }
}
